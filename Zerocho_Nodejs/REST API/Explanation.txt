GET : 서버 자원을 가져오고자 할 때  사용    요청의 본문(body)에 데이터를 넣지 않는다
      데이터를 서버로 보내야한다면 쿼리스트링을 사용

POST : 서버에 자원을 새로 등록하고자 할 때 사용함 
       요청의 본문에 새로 등록할 데이터를 넣어 보내야한다면

PUT : 서버의 자원을 요청에 들어 있는 자원으로 치홚고자 할 때 사용
      요청의 본문에 치환할 데이터를 넣어 보냅니다.

PATCH : 서버 자원의 일부만 수정하고자 할 때 사용 
        요청의 본문에 일부 수정할 데이터를 넣어보냅니다

DELETE : 서버의 자원을 삭제하고자 할 때 사용
